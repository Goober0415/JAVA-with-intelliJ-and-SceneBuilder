# JAVA I — CNM Course Projects

A collection of Java projects completed for JAVA I at Central New Mexico Community College. Projects are built with IntelliJ IDEA and, where applicable, Scene Builder for JavaFX UI design.

## 🛠️ Tools & Technologies

- **Language:** Java
- **IDE:** IntelliJ IDEA
- **UI Design:** Scene Builder (JavaFX)
- **JDK:** JDK 23

## 📁 Projects

Each project lives in its own folder within this repo:

| Project | Description |
|---|---|
| `TextAreaDemoDowdenDuarte` | Swing demo using a custom `DescriptionPanel` component with `JTextArea`, `JLabel`, and image display |
| *(more projects coming as the course progresses)* | |

## 🚀 Running a Project

1. Open IntelliJ IDEA
2. Select **File → Open** and navigate to the project folder
3. Make sure the JDK is set to JDK 23 under **File → Project Structure → SDK**
4. Place any required resource files (images, etc.) inside the `src` folder
5. Run the `main` method in the main class

## 📋 Notes

- Resource files (like images) must be placed in the `src` directory so IntelliJ includes them on the classpath
- Scene Builder `.fxml` files are linked to their controllers via IntelliJ's JavaFX configuration
